Box-model Topic

A CSS box model is a compartment that includes numerous assets, such as edge, border, padding and material. It is used to develop the design and structure of a web page. It can be used as a set of tools to personalize the layout of different components. According to the CSS box model, the web browser supplies each element as a square prism


1.border : It is a region between the padding-box and the margin. Its proportions are determined by the width and height of the boundary.


2.margin : This segment consists of the area between the boundary and the edge of the border. The proportion of the margin region is equal to the margin-box width and height. It is better to separate the product from its neighbor nodes


3.padding : This field requires the padding of the component. In essence, this area is the space around the subject area and inside the border-box. The height and the width of the padding box decide its proportions


4.content : Material such as text, photographs, or other digital media is included in this area.It is constrained by the information edge, and its proportions are dictated by the width and height of the content enclosure.

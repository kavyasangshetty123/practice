Heading Tag

The h1 to h6 HTML elements represent six levels of section headings. h1 is the highest section level and h6 is the lowest.

The H1 tag is an HTML heading that's most commonly used to mark up a web page title.
An H2 tag marks the first sub-heading after your document's main heading. It defines the second-level headings on your webpage.
The H3 element defines a level-three heading.
The H4 element defines a level-four heading.
The H5 element defines a level-five heading.
